Hello, I'm Aayush 👋

I'm a third-year Information Technology engineering student with a deep curiosity for how software systems are built, deployed, and scaled. My journey into tech started with tinkering on old computers, and it quickly evolved into a passion for open-source tools, systems programming, and cloud infrastructure.

This blog is a mini-project for my Cloud Computing course, but it represents something more to me — a place to document what I learn and share it with others. Every post here is something I've genuinely explored and found interesting enough to write about.

Outside of academics, I spend time building small games, reading about Japanese culture and language, and exploring the rabbit holes that Linux inevitably leads you down. I believe the best way to learn something is to build with it — which is exactly what this project is about.

📌 About This Project

This blog is built with plain HTML & CSS, version-controlled on GitHub, and automatically deployed to Azure Static Web Apps via a GitHub Actions CI/CD pipeline. It demonstrates core cloud computing concepts — static hosting, CDN delivery, and automated deployment — as part of the third-year IT curriculum.
